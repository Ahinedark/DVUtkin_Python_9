# Домашнее задание к лекции 5 "ООП". Уткин Д.В.

Реализовать скачивание с loremflickr.com в асинхронном режиме не удалось, т.к. при выполнении команды "await response.read()" зависает намертво. Оказалось, что это особенность работы aiohttp с этим сайтом.  
Удалось реализовать скачивание с placebear.com, но там не такое большое обилие картинок (и чтобы картинки выходили разные, нужно в запросе указывать разные размеры).  

Скрипт скачивает 10 (5+5) картинок в директорию DVUtkin_Python_9/downloads.  
Содержит две реализации загрузки:  
1. Синхронная с использованием requests.  
2. Асинхронная с использованием aiohttp и aiofiles.  

Выводит в консоль время выполнения обеих реализаций.
